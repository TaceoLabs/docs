# Roadmap

The TACEO Network is evolving toward a more distributed and scalable infrastructure for confidential computation.

Key areas of development include:

**Expansion of Node Operators**

Increasing the number and diversity of independent infrastructure operators participating in the network. This also includes focus on geographic distribution of MPC nodes.

**Hardware-based attestation**

While the guarantees of the TACEO Network are based on multiparty computation protocols, we are exploring the use of confidential computing technologies such as AMD SEV-SNP and Intel TDX. This allows nodes to attest that execute approved software and allows clients to verify that they are interacting with a genuine MPC node.

**Improved developer integrations**

Expanding APIs and tooling to simplify integration of privacy services into applications.

These developments aim to make confidential computation infrastructure more robust, accessible, and scalable for real-world applications.
